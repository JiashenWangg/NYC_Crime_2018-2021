## NYC Crime Analysis (2018–2021)

Data visualization and exploratory analysis project on New York City crime patterns using NYPD data.

### Overview

This project analyzes crime data in New York City from 2018 to 2021 to understand spatial distribution, temporal trends, and demographic characteristics.

The analysis highlights borough-level differences and the impact of COVID-19 on crime trends.

### Dataset

Source: https://www.kaggle.com/datasets/supreeth888/nypd-data

Time period: 2018–2021

Size: ~775,000 records

Includes:

Crime type and severity

Borough and location (latitude/longitude)

Victim and suspect demographics

### Key Questions

How are crimes distributed across NYC?

How did crime trends change over time (especially during COVID-19)?

What are the characteristics of victims and suspects?

### Methods

Data cleaning and preprocessing in R

Exploratory Data Analysis (EDA)

Statistical testing (Chi-squared test)

### Visualization:

Time series plots

Geospatial heatmaps

Mosaic plots

Correlation plots

Word clouds

### Key Findings

Crime is concentrated in Manhattan, Brooklyn, and the Bronx

Residential locations have the highest crime frequency

Crime dropped significantly during early COVID-19 and rebounded in 2021

Most victims fall in the 25–44 age group

Victim and suspect demographics show strong correlations

### Future Work

Analyze crime by specific categories

### Summary

This project demonstrates how data visualization and statistical analysis can uncover meaningful insights into urban crime patterns and support data-driven decision-making.
